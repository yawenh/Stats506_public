## Activities:

### Week1:

* `week1_part1.sh` - a shell script used to download [RECS](https://www.eia.gov/consumption/residential/data/2015/index.php?view=microdata) data and extract columns by names.
* `week1_part2_args.sh` - Illustration for how to use arguments passed at the command line.
* `cutnames.sh` - Modified version of week1_part1.sh into an executable program.
  
### Week4:

* `week4_part1.Rmd`(dplyr) - an Rmd file used to explore or analyze the `Orange` data from R's `datasets` package. 
* `week4_part2.Rmd`(ggplot2) - an Rmd file that use `w4_p2_RECS2015_internet.csv` data to reproduce plot `w4_p2_q1_plot.png` and `w4_p2_q2_plot.png` in each question using ggplot.
* `w4_p2_RECS2015_internet.csv` - dataset used for part2.
* `w4_p2_q1_plot.png` and `w4_p2_q2_plot.png` - sample plots need to be reproduced.
